# F For Furniture

A shopping website if you want to buy furniture. It has a nice CSS only slider and icon for the tab and flex cards. I found this while I was searching for the [Arlecchino Tribute Page](https://raghacchino.github.io/arlecchino-tribute-website/) on Pinterest. I liked it so I thought I could make it. 

## Features
- A CSS only carousel on the hero section
- A grid layout but without using display grid (using flexbox)
- An about us page with text that has an img as background
- A contact us page
- A footer
