# f-for-furniture

A shopping website if you want to buy any furniture. It has a nice CSS only slider and icon for the tab and flex cards. I found this while I was searching for the [Arlecchino Tribute Page](https://raghacchino.github.io/arlecchino-tribute-website/) on Pinterest. I liked it so I thought I could make 2 or 3 sections of it. The carousel was a little tricky but everything else was straight forward. The categories cards could be even simpler if I knew how to use display grid but I just did it with display flex.

Design inspiration linked here [pin](https://in.pinterest.com/pin/1122311169750452572/)
